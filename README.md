# 📚 Smart Study Planner

## 📌 Problem
Students often struggle to manage their study tasks and deadlines effectively, which leads to poor time management, missed deadlines, and increased stress.

## 💡 Solution
The Smart Study Planner is a simple and user-friendly web application that helps students organize their daily tasks, set priorities, and track their progress efficiently.

## ✨ Features
- Add tasks easily
- Delete tasks
- Mark tasks as completed
- Set priority levels (High, Medium, Low)
- Set deadlines for tasks
- Filter tasks (All / Completed / Pending)
- Data is saved using localStorage (remains even after refresh)

## 🛠️ Technologies Used
- HTML (Structure)
- CSS (Styling)
- JavaScript (Functionality & Logic)

## ▶️ How to Run
1. Download or clone this repository
2. Open the `index.html` file in any web browser
