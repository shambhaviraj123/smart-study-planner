# 📚 Smart Study Planner

## 📌 Problem
Students often struggle to manage their study tasks and deadlines effectively, which leads to poor time management and stress.

## 💡 Solution
The Smart Study Planner is a simple web-based application that helps students organize their daily tasks, set priorities, and track their progress efficiently.

## ✨ Features
- Add tasks
- Delete tasks
- Mark tasks as completed
- Set priority (High, Medium, Low)
- Set deadlines
- Filter tasks (All / Completed / Pending)
- Data saved using localStorage (remains after refresh)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## ▶️ How to Run
1. Download or clone this repository
2. Open the `index.html` file in any web browser
